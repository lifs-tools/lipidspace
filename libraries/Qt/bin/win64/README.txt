LipidSpace


Installation:
No installation required, just unzip the zip folder whereever you want.


Starting LipidSpace:
In the LipidSpace folder, just click on LipidSpace.exe


How to use LipidSpace:
You'll find four interactive tutorials on the 'Home' tab of the tool directly after launching. For more information, visit:
https://lifs-tools.org/lipidspace.html
